Plant Parenthood

Frameworks: Express & Partials



The primary features of this website is to record data in regards to plant keeping. Using the 7 restful routes CRUD (create, read, update delete), a user can view plant entries, add new plants, edit current plant entries, and delete plant entries. The app also uses Auth so a user must create a login to interact with the site to use the create, read, update and delete features.
